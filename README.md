# PartsBrain AI — 部品流通DXプラットフォーム

## セットアップ & 起動

```bash
npm install
npm run dev
```

## Vercel デプロイ

```bash
npm install
npx vercel --prod
```

## 画面一覧

| 画面 | 説明 |
|------|------|
| ダッシュボード | KPI・受注サマリー・AIアクションログ |
| AI部品診断 | 画像スキャン → 部品特定 → クロスセル提案 |
| 動的プライシング | SHAP分析・価格シミュレーター・競合比較 |
| マルチ見積もり | RFQ一覧・仕入先比較テーブル・ワークフロー |
| 部品マスタ | 部品登録・在庫・粗利率管理 |
| 仕入先マスタ | 仕入先登録・評価・取引条件管理 |
| 顧客マスタ | 顧客登録・価格グループ・売上管理 |
| 受発注管理 | 受注ステータス管理・AI手配連携 |
